# HTML,JavaScript,CSS,jQueryの勉強の第一弾

## 句読点を変換してくれるサービス

## 概要
- 大学の講義などで、全角の「、。」などは半角の「,.」に変換する必要がある
- そのような処理を自動で行ってくれるサービス
- 変換をすると文字数、行数も同時にカウントしてくれる

## 例
- 入力
> 今日は、いい天気ですね。
- 出力
> 今日は,いい天気ですね.
